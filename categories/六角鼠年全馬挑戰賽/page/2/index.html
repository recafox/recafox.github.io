Git 指令

  1. git log 查詢過往commit紀錄
    (按q退出)

  2. git branch 看目前分支

  3. 還原前版
    git checkout 要前往的過去版號四碼
    git checkout master 確定還原

  4. HEAD 目前所在位置的指標

  5. git branch new_branch_name 新增分支

    git branch 查詢目前有的分支, 目前所在分支前面會有星號
    git checkout new_branch_name 切換到新增的分支
    git checkout master 切回主枝

  6. git status 目前分支的狀態, 檔案修改紀錄

  7. git merge new_branch
    將新增分支中的更新也套用至master

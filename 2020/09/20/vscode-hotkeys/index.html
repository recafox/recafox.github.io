VS code 常用快捷鍵 (for windows)

整理了一些vs code常用快捷鍵。

文字編輯區

注意: 有些指令在按下第一次後, 再按第二次時可以還原

快捷鍵 用途
ctrl + z 還原
ctrl + shift + z 取消還原
ctrl + /, ctrl + k + c 單行註解
ctrl + /, ctrl + k + u 還原單行註解
alt + shift + a 多行註解
alt + 上/下鍵 向上/下移動選取區塊
shift + alt + 上/下鍵 複製目前這一行並貼在下方, 不用選取
shift + ctrl + k 刪除目前這一行, 不須選取
shift + alt + f 文件格式化(根據目前設定的格式)
ctrl + +/- 放大/縮小字體
alt + z 單行字數太多時自動換行
shift + 方向鍵 以目前滑鼠所在的位置, 向上/下/左/右選取
ctrl + h 在目前編輯視窗中進行替換

工具型

快捷鍵 用途
ctrl + b 顯示/隱藏左側工具列
ctrl + shift + n 新增一個編輯視窗
ctrl + w 關閉目前所在編輯視窗
ctrl + k + w 關閉目前所有編輯視窗
f11 全螢幕化
ctrl + b 顯示/隱藏左側工具列
ctrl + p 以檔名或檔案類型尋找檔案
ctrl + shift + f 全域尋找檔案, 可以使用檔名, 副檔名, 或是匹配該檔案中的文字

終端機

快捷鍵 用途
ctrl + ` 打開terminal
ctrl + shift + ` 新增terminal
ctrl + shift + 5 分割終端機頁面

什麼是函式？

函式本身是物件下的子類別，跟物件有區別。
函式可以被呼叫，裡面包含程式碼的片段而不是屬性和值

當使用function這個詞來宣告函式時，該程式碼就會成為函式，這個函式可以被呼叫，裡面包含程式碼片段，也會有名稱（但不一定，若沒有名稱的話就是匿名函式）

立即函式 IIFE

IIFE: Immediately Invoked Function Expression
立即呼叫的函式表達式
也被稱為會自我呼叫的匿名函式

以小括號包住一個函式的陳述式，宣告完就會立即執行，無法在函式外被再次執行
